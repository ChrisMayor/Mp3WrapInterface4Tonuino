# Mp3WrapInterface4Tonuino
Interface for mp3wrap to convert mp3s to correct format for the Tonunio https://www.voss.earth/tonuino/

Requirements:

* .NET Framework 4.7.2
* mp3wrap (http://mp3wrap.sourceforge.net/)
